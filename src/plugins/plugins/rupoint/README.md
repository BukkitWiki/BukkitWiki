## 简介
:::tip 适用于绝大部分属性加点场景 
:::

高度自定义的属性加点插件
- 自定义GUI
- 自定义加点属性（可多属性）
- 支持AP、SX（需要添加其他属性插件可以联系作者添加）
- 自定义事件给玩家添加属性点
- 支持暗改某个玩家下次加点一定成功
