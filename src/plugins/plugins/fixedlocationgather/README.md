## 简介

一个功能完备的固定地点采集插件
::: info QQ交流群 642468469 
:::


## 功能特点

- 支持多个采集点配置,可自定义位置范围和采集要求
- 完善的采集进度显示,包括进度条、动作条、Boss血条等多种方式
- 内置体力系统,合理限制采集频率
- 灵活的时间管理,可设置采集点开放时间段
- 资源系统支持,包括资源消耗、恢复等机制
- 丰富的奖励机制,支持物品、命令、脚本等多种奖励方式
- DragonCore界面支持,提供更好的视觉体验
- 完整的权限管理和消息提示系统
- 支持JS和Kether脚本,实现自定义检查和奖励逻辑
