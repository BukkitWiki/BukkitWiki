### 简介

::: info 这个插件可以胜任绝大多数延时制作物品的场景，例如：作物生产、物品制作、物品增强。
:::

QQ交流群： 642468469

特性：
 - 可以创建无限个GUI
 - 高度自定义 在下面视频中所有你看到的内容都是可自定义的
 - 内置功能强大的脚本系统


### 配置

配置文件结构如下：

- [farm(农场配置文件夹)](./farm/)
- config.yml 主配置文件夹
- [seed.yml 种子配置](./seed/)
- [soil.yml 土壤配置](./soil/)
